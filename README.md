# ai_movie_assistant
Assistente de criação de curtas de IA 

Resumo: Automação de agentes de IA que seguem uma corrente de produção audiovisual.

Problema a resolver: Automatizar produção audiovisual à partir de apenas um prompt com ideia inicial.

Ferramentas utilizadas
- Python/JavaScript
- APIs de filmes
- OpenAI API / LLMs
- Pandas

Aprendizados:

- Como integrar múltiplas APIs em um único fluxo
- Construção de um sistema de recomendação baseado em contexto
- Aplicação de engenharia de prompt para melhorar sugestões
- Estruturação de dados para alimentar modelos de IA
- Manipulação de dados externos (filmes, gêneros, avaliações)
- Criação de uma experiência de usuário simples com IA
- Tratamento de erros e respostas inconsistentes de APIs
- Organização de projeto voltado para produto (não só script)
